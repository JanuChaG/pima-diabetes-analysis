# pima-diabetes-analysis
This is for the Coursework 2 of Principles of Data Science module of MSc in Data Science/Coventry University

This project predicts diabetes risk using the Pima Indians Diabetes dataset.  
It includes exploratory analysis, clustering, predictive modeling, and a short cloud integration overview. 

## 📝 Documentation
See [DECISIONS.md](DECISIONS.md) for detailed notes on assumptions, model choices, challenges, and workflow decisions.
Environment reproducibility is ensured through [requirements.txt](./requirements.txt), generated directly from the notebook environment.
